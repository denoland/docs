New to Deno? Install the runtime, run your first script, and set up your editor.
Once you're up and running, use the second-level navbar to dive into a specific
job — running code, managing packages, testing, and more.

## Install and run

- **[Installation](/runtime/getting_started/installation/)** — install Deno on
  macOS, Windows, or Linux.
- **[First project](/runtime/getting_started/first_project/)** — write and run a
  small program end to end.
- **[Set up your environment](/runtime/getting_started/setup_your_environment/)**
  — configure your editor, shell completions, and language server.
- **[Command line interface](/runtime/getting_started/command_line_interface/)**
  — a tour of the `deno` CLI and its most common subcommands.

## Where to next

- **[Run code](/runtime/run/)** — execute scripts, serve HTTP, run tasks.
- **[Manage packages](/runtime/packages/)** — add dependencies, use npm, publish.
- **[Migrate to Deno](/runtime/migrate/)** — coming from Node.js or Deno 1.x.
- **[Reference](/runtime/reference/)** — look up CLI commands, config, and APIs.
