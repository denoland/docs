---
title: "Build CLI apps"
description: "Compile your Deno program into a single self-contained executable and distribute command-line tools."
---

Turn a Deno program into a standalone, self-contained executable with
`deno compile` — no runtime to install on the target machine — or distribute it as
an installable command-line tool.

## Reference

- **[deno compile](/runtime/reference/cli/compile/)** — build a single-file
  executable for any supported target platform.
- **[deno install](/runtime/reference/cli/install/)** — install a script as a
  globally available command.

## Related

- **[deno task](/runtime/reference/cli/task/)** — define and run project scripts.
- **[Desktop apps](/runtime/desktop_apps/)** — building graphical desktop
  applications.
