---
title: "Test your code"
description: "Write and run tests, measure coverage, and benchmark performance with Deno's built-in test runner."
---

Deno ships a test runner, coverage, and a benchmarking tool — no dependencies to
install. Write tests with `Deno.test`, run them with `deno test`, and measure
coverage and performance.

## Guides

- **[Testing](/runtime/fundamentals/testing/)** — write tests, assertions, mocks,
  snapshots, and behavior-driven tests.

## Reference

- **[deno test](/runtime/reference/cli/test/)** — run your test suite.
- **[deno bench](/runtime/reference/cli/bench/)** — benchmark code with
  `Deno.bench`.
- **[deno coverage](/runtime/reference/cli/coverage/)** — generate and report
  test coverage.
