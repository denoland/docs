## Logo component

Size constrained by its containing element

```tsx
<comp.Logo />
```

## Search input

Form element for searching the deno domain with google search

```tsx
<comp.SearchInput />
```

## External link callout

Decorator for external links

```tsx

  deno.com
</comp.ExternalLink>
```
